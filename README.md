# ClearThePoints

## Giới thiệu

ClearThePoints là một trò chơi đếm số, nơi người chơi cần phải chọn các số theo thứ tự.

## Chạy chương trình

B1: git clone https://github.com/NguyenQuocTien240103/UpgradeClearThePoints.git

B2: cd ./vite-project

B3: npm install

B4: npm run dev
